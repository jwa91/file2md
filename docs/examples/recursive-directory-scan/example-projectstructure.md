# Example Project Structure

This is a project structure for demonstrating recursive directory scanning:

```
my-project/
├── README.md
├── file2md/
│   ├── .file2mdignore
│   ├── files.txt
│   └── output.md
├── package.json
├── src/
│   ├── index.js
│   ├── components/
│   │   ├── Button.js
│   │   ├── Card.js
│   │   └── Input.js
│   └── utils/
│       ├── formatting.js
│       ├── validation.js
│       └── helpers.js
└── test/
    ├── unit/
    │   ├── components/
    │   │   └── Button.test.js
    │   └── utils/
    │       └── validation.test.js
    └── integration/
        └── api.test.js
```
