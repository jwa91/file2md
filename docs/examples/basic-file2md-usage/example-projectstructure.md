# Example Project Structure

This is a simple project structure for the basic file2md usage example:

```
my-project/
├── README.md
├── file2md/
│   ├── .file2mdignore
│   ├── files.txt
│   └── output.md
├── package.json
├── src/
│   ├── index.js
│   ├── config.js
│   └── utils.js
└── test/
    └── index.test.js
```
