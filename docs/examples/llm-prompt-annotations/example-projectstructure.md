# Example Project Structure

This is a project structure for demonstrating how to use file2md to include instructions and prompts for an LLM:

```
my-project/
├── README.md
├── package.json
├── file2md/
│   ├── .file2mdignore
│   ├── files.txt
│   └── output.md
└── src/
    ├── index.js
    ├── api.js
    ├── config.js
    ├── utils.js
    └── components/
        ├── Button.js
        ├── Card.js
        └── Input.js
```
